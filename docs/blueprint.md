# **App Name**: PromptVibes

## Core Features:

- Learning Roadmap: Interactive Learning Roadmap: Display learning modules categorized under Beginner, Intermediate, and Professional levels, navigable through tabs.
- Dynamic Level Content: Dynamic Routing: Implement dynamic routes for topics based on the learning level selected by the user (/learn/[level]).
- Vibe UI Components: Reusable UI Components: Design and integrate reusable components like cards, tabs, badges, and accordions from Vibe UI for consistent design.
- Prompt Analysis: AI Prompt Analyzer: Analyze the quality of prompts crafted in the playground and offer suggestions for improvement using an AI tool.

## Style Guidelines:

- Primary color: A saturated blue (#29ABE2) to invoke a sense of clarity and focus.
- Background color: Off-white (#F9F9F9), nearly desaturated blue, for a clean and distraction-free interface.
- Accent color: A vibrant green (#90EE90) for highlighting key interactive elements and CTAs.
- Body and headline font: 'Inter' sans-serif for a modern and neutral aesthetic.
- Minimalist icons to represent different learning modules and actions.
- Clean and spacious layout with a focus on readability and ease of navigation.
- Subtle animations and transitions to enhance user experience and provide feedback on interactions.
